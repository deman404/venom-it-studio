Venom IT is a cutting-edge studio synthesizing high-performance tech with predatory bio-organic aesthetics.
Specializing in prological architectures and lethal game engines, we inject adaptive logic into the digital substrate.
The interface leverages immersive neon accents, fluid slime textures, and morphing shapes to create an intimidating presence.
Integrated Gemini AI powers a "Toxic Assistant" for real-time neural sync and interactive bio-logical consulting.
We redefine digital biology by architecting self-evolving systems designed to dominate the future of the network.


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
